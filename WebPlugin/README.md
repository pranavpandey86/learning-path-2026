# Product Widget

A self-contained Angular web component for displaying a product catalog with ordering capability.

## Quick Start

```bash
npm install
npm run build:elements
```

Output is in `dist/product-widget/`

## Integration

See **[INTEGRATION_GUIDE.md](./INTEGRATION_GUIDE.md)** for complete step-by-step instructions.

## Development

```bash
npm start
# Open http://localhost:4200
```
